## Installation
Clone repository

Dependencies
- Vite
- React

Avec pnpm :
```shell
pnpm i
npm run develop
```

Avec npm :
```shell
npm i
npm run develop
```

Avec yarn :
```shell
yarn
yarn develop
```

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
